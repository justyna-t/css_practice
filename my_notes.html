Information about HTML

What is HTML?

HTML stands for HyperText Markup Language. HTML is the language used o create webpages. HTML is a main document type of the web, basic for almost every web page. HyperText Markup Language have specific syntax and rules and tells a browser how to display content on a web page.

HTML is made up of:
  • text content (what I see)
  • markup (what it looks like)
  • references to other documents (eg. videos, images)
  • links to other pages

HTML elements

We use HTML elements to tell a browser how to display content on a web page.

Most HTML elements looks like:
opening tag + tag contents + closing tag = element (tag)
<NAME>Contents</NAME>

We can divide HTML elements into inline elements and block elements.

Inline elements:
  • <b>content</b> Makes content bold
  • <em>content</em> Makes content italic
  • <img> Adds image
  • <a>link</a> Used to make a link
  • <br> Creates a break line.
  • <span>text</span> Provides no visual change by itself (need to use attributes).
  • <strong>text</strong> Defines important text.
Block elements:
  • <p>text</p> Defines a paragraph.
  • <div></div> Defines a division/ a section in an HTML document.
  • <form></form> Used to create an HTML form for user input.

HTML attributes

<TAG attribute="value">content</TAG>

Some HTML elements accept additional values called attributes in their opening tags. Attributes provide additional information to the browser.

Examples:
<a href="url">link</a>
Creates a hyperlink to the href attribute value url and names it link .
<img src="url" alt="text">
Displays the image hosted at the src attribute value "url". The alt attribute value "text" will be shown when the image cannot be displayed. Value "url" can also be a path to local file on my computer.

Information about CSS

Introduction to CSS

DOM
Document Object Model. When programmers talk about the "DOM" they are talking about the tree-like structure of a page. DOM is a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages).
The nodes of every document are organized in a tree structure, called the DOM tree. Each HTML tag creates an element in the DOM that the browser uses to display the page. The HTML document can provide the browser with information on what style to use and where to get it.
CSS
Cascading Style Sheets. CSS is a style sheet language used for describing the look and formatting of a document written in a markup language.
CSS have specific syntax and rules to change how elements will look on the page. For example: the size of the font, the background, color.

Understanding CSS

You can also think about CSS as a search and replace tool: you identify a class or a tag of the element you want to find (or match, in CSS terminology), and then what you want to do with it, or what property values to replace with different ones.

"Cascading" means that rules are applied not only to the elements they directly match, but also to all of those elements child. If a child element has multiple, overlapping rules defined for it, the more specific rule takes effect.

How to use CSS?

It also matters where you define the rules and in what order they are applied. Styles can be defined in different places and are applied in the following order, with definitions further down the list overwriting previous definitions:
  • the default style of a browser (different browsers have slightly different styles)
  • stylesheet in a separate CSS file (external, best way)
  • stylesheet inside HTML (internal, not ideal)
  • style in an element (inline, should be avoided)

We can use !important. !important always will be first independently of other rules.

Example: outline: 1px solid red !important

Write your style inline with your HTML.

This is generally a very bad idea because it leads to lots of repeated code but it's good to be familiar with it.

With this method, you modify the style attributes of every individual HTML element.

Box sizing

Everyting on a web page is a box.

There are four main points about box sizing:
  1. HTML elements are boxes and each box has 4 components.
  2. It can often be hard to get the size of a box just right, because there are so many components to each box.
  3. There are two techniques to use to help deal with sizing issues:
    1. Set sizes in terms of percentages rather than pixels.
    2. Set the box-sizing attribute to border-box for every element.
  4. Different browsers work slightly differently. Sometimes this causes different browsers to display the same code differently.

Four components of box are: margin, border, padding and content.

More about box-sizing:
box-sizing: content-box OR border-box
content-box is default. The width and height includes ONLY content.
border-box - The width and height includes content, padding and border WITHOUT margin
Browsers have diffrent styles, so is important to use browser-specific prefixes:
  • -webkit-box-sizing: border-box (webkit for Google Chrome, Safari and Opera)
  • -moz-box-sizing: border-box (moz for Mozilla Firefox)
  • -ms-box-sizing: border-box (ms for Microsoft Internet Explorer)

Box positioning

  1. Divs are block elements (as opposed to inline), so by default they take up the entire width of a page.
  2. Adding the rule display: flex; to the appropriate CSS will override this behavior and let divs appear next to each other.

Flexible Box Layout module, flex items in flex container.

Great guide to flexbox here: flexbox

Properties for the Parent(.container):
  • {display: flex;} the shorthand for flex-grow, flex-shrink and flex-basis
  • {flex-directon: row | row-reverse | column | column-reverse;}
  • {flex-wrap: nowrap | wrap | wrap-reverse;}
  • {justify-content: flex-start | flex-end | center | space-between | space-around}
  • {align-items: flex-start | flex-end | center | baseline | stretch;}
  • {align-content: flex-start | flex-end | center | space-between | space-around | stretch;}
Properties for the Children (.item):
  • {order: <integer>}
  • {flex-grow: <number>; /* default 0 */ }
  • {flex-shrink: <number>; /* default 1 */ }
  • {flex-basis: <lenght> | auto; /* default auto */}
  • {align-self: auto | flex-start | flex-end | center | baseline | stretch}
